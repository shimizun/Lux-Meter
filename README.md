# Wio Terminal High-Performance Lux Meter 💡

Wio Terminalと高精度照度センサー **VEML7700** を使用した、リアルタイム照度計（ルクスメーター）です。
人間の目の特性に近い**対数スケール（Log Scale）**でのトレンドグラフ表示と、スプライト（ダブルバッファリング）技術による滑らかなUIを実現しています。



## 🚀 主な機能

* **高精度測定**: VEML7700を使用し、広範囲な照度（lux）を正確に取得。
* **オートレンジ機能**: 周囲の明るさに合わせて、センサーのゲイン（感度）を自動的に最適化（ヒステリシス制御付き）。
* **対数トレンドグラフ**: 直近320サンプルの履歴を表示。対数スケールを採用しているため、暗い場所から直射日光下まで直感的に把握可能。
* **ダブルバッファリング**: `TFT_eSprite` を活用し、画面更新時のチラつきを完全に排除。
* **メモリ最適化**: カラー深度を8bitに調整することで、Wio Terminalの限られたRAM内でも安定動作を実現。

## 📚 依存ライブラリ

Arduino IDEのライブラリマネージャーから以下のライブラリをインストールしてください。

1.  **Seeed_Arduino_LCD** (Wio Terminal 標準ライブラリ)
2.  **Adafruit VEML7700 Library** (センサー制御用)
3.  **Adafruit BusIO** (依存ライブラリ)

## 💻 使い方

1.  Wio Terminalの右側にあるGroveポート（I2C）にVEML7700を接続します。
2.  本リポジトリのスケッチをArduino IDEで開き、ボードを「Wio Terminal」に設定してコンパイル・書き込みを行います。
3.  起動後、即座にリアルタイム測定が開始されます。

### 画面の見方

* **メイン数値**: 現在の照度（lux）をリアルタイム表示。
* **モード表示（右上）**: 現在のセンサーゲイン（High Sens / Wide Range）を表示。
* **下部グラフ**: 過去の照度推移。0.5秒間隔で更新されます。



## 🔍 技術解説：対数（Log）グラフの利点

人間の視覚は、明るさの変化を線形ではなく「対数的」に捉える特性があります。本プロジェクトでは以下の計算式を用いてグラフを描画しています。

$$y = \text{GRAPH\_H} - \left( \log_{10}(\text{lux}) \times \frac{\text{GRAPH\_H}}{4} \right)$$

これによって、1 lux（暗い部屋）から 10,000 lux（明るい直射日光）までの劇的な変化を、ひとつのグラフ内で潰れることなく視覚化しています。
